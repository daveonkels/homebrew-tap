# Homebrew Tap for daveonkels

This is a [Homebrew](https://brew.sh/) tap for installing CLI tools and applications by [@daveonkels](https://github.com/daveonkels).

## Available Formulae

### clinkding

Modern CLI for [linkding](https://github.com/sissbruecker/linkding) bookmark manager.

**Installation:**

```bash
brew tap daveonkels/tap
brew install clinkding
```

**Usage:**

```bash
# Configure
clinkding config init

# List bookmarks
clinkding bookmarks list

# Get help
clinkding --help
```

## About Homebrew Taps

A tap is a repository of formulae for Homebrew. This tap provides formulae that aren't in the main Homebrew repository.
